# dlai

Minimal Python project for working with OpenAI APIs and prompt parsing.

## Prerequisites

- Python 3.14+
- OpenAI API key

## Setup

```bash
# Create venv and install dependencies (uv)
uv sync

Create a `.env` file with your API key:

```
OPENAI_API_KEY=your-key-here
```

## Usage

```bash
python main.py
```

For prompt parsing experiments, use the `L1-Model_prompt_parser.ipynb` notebook.
